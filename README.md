# Productividad con IA - CMS

Blog CMS sobre productividad con inteligencia artificial. Construido con Next.js 14 + Payload CMS 2 + Tailwind CSS, desplegado en Vercel.

## Cuatro pilares de contenido

| Slug | Categoria |
|---|---|
| ia-profesiones | IA para Profesiones |
| automatizacion-workflows | Automatizacion de Workflows |
| libreria-prompts | Libreria de Prompts |
| resenas-herramientas | Resenas de Herramientas |

## Stack

- Frontend: Next.js 14 (App Router, ISR)
- CMS: Payload CMS 2 (panel admin en /admin)
- DB: MongoDB
- Estilos: Tailwind CSS
- Deploy: Vercel
- Automatizacion futura: n8n / Make + Gemini API

## Variables de entorno

Copia .env.example a .env.local y rellena los valores.

## Desarrollo local

npm install
npm run dev

- Frontend: http://localhost:3000
- Panel CMS: http://localhost:3000/admin
